💱 Currency Converter Web App

A sleek and user-friendly Currency Converter that allows users to convert currencies effortlessly. Built with HTML, CSS, and JavaScript, this web app fetches real-time exchange rates and provides an intuitive interface for quick conversions.

✨ Features

✅ Easy-to-use currency selection with country flags
✅ Live exchange rate updates
✅ Stylish and responsive design
✅ Smooth UI with interactive elements

🛠️ Technologies Used

HTML5 for the structure
CSS3 for styling
JavaScript (ES6+) for dynamic functionality
FontAwesome for icons
📂 Project Structure

/currency_converter
│── currency_index.html   # Main interface
│── currency_style.css    # Styling
│── currency_script.js    # Main logic for currency conversion
│── codes.js              # Additional scripts
🚀 How to Use

Enter the amount you want to convert.
Select the currencies (From & To).
Click "Get Exchange Rate" to see the converted value.
